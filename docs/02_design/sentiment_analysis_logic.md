# FinShift Sentiment Index Design

FinShift独自の市場センチメント分析ロジックと、それがユーザー（スイングトレーダー）に提供する価値についての定義書です。

## 1. なぜ独自開発するのか？ (Why Custom?)

CNN "Fear & Greed Index" などの既存指標を利用せず、独自の指標「FinShift Sentiment Index」を算出する理由は以下の通りです。

1.  **「ニュースの空気感」を反映させるため**:
    - 既存の指標はVIXやモメンタムなどの「テクニカル（数値）」のみで構成されることが多いです。
    - しかし、実際の市場は「数値はまだ動いていないが、ニュースで懸念が広がり始めている（＝嵐の前の静けさ）」という局面があります。
    - FinShiftでは、**Geminiによるヘッドライン分析（AI News Sentiment）** を組み込むことで、この**「予兆」**をスコアに反映させます。

2.  **権利関係のクリア化**:
    - CNN等の指数はプロプライエタリ（商標・知的財産）であり、無断で商用サイトに掲載・再配信することはリスクがあります。
    - 自社計算であれば、完全に自由にウィジェット化・API化が可能です。

3.  **スイングトレード特化の調整**:
    - 一般的な指標は「長期投資家」向けの場合があります。
    - FinShiftは「数日〜数週間」のスイングトレーダー向けに、**実際の価格変動（Momentum）** と **恐怖指数（VIX）** の比重を調整し、より敏速なシグナルを出します。

---

## 2. 算出ロジックの概要 (Logic)

**FinShift Sentiment Index (0-100)**
- **0 - 25**: **Extreme Fear** (パニック、総悲観)
- **26 - 45**: **Fear** (警戒、リスクオフ)
- **46 - 54**: **Neutral** (様子見)
- **55 - 75**: **Greed** (楽観、リスクオン)
- **76 - 100**: **Extreme Greed** (陶酔、過熱感)

### 構成要素
| 要素 | 比重 | ソース | 役割 |
| :--- | :--- | :--- | :--- |
| **VIX (Volatility Index)** | 40% | `market_data.py` | 恐怖のベースライン。数値が高いほどスコアを下げる（逆相関）。 |
| **Market Momentum** | 20% | `S&P500 1W Change` | トレンドの方向性。上昇トレンドならスコアを上げる。 |
| **AI News Sentiment** | 40% | `Gemini Analyzed Headlines` | **強み**: ニュースの文脈から「数値には表れていないリスクや期待」を読み取る。 |

## 3. UI表示場所 (Where to Display)

1.  **トップページ (Dashboard)**:
    - ファーストビューにメーター（ゲージ）として表示。
    - ユーザーがサイトを開いた瞬間、**「今の相場の天気（晴れか嵐か）」** を直感的に把握できます。
    - [Wireframe: Front Page](../02_design/wireframes/front_page.md) 参照。

2.  **Daily Briefing 記事ヘッダー**:
    - 各国の市況まとめ記事の冒頭に配置。
    - 「今日のセンチメント: Extreme Greed (85)」として表示し、記事を読む前のコンテキストを作ります。

---

## 4. ユーザーへの提供価値 (User Value)

この指標を見ることで、スイングトレーダーは以下のような **「逆張り（Contrarian）」** または **「順張り（Follow）」** の高度な意思決定が可能になります。

### ケーススタディ

| Sentiment State | ユーザーの解釈・アクション |
| :--- | :--- |
| **Extreme Fear (<25)** | **「買い場を探す (Buy the Dip)」** <br> 大衆がパニックで投げ売りしている底値圏の可能性がある。AIシナリオが「Bullish（反発）」を示していれば、絶好のエントリーチャンス。 |
| **Fear (26-45)** | **「慎重な押し目買い / 空売りの手仕舞い」** <br> 警戒感はあるが、パニックではない。リスク管理を徹底しつつ打診買いの準備。 |
| **Neutral (46-54)** | **「方向感が出るまで待つ (Wait & See)」** <br> 無理にエントリーせず、AIシナリオの「重要ラインブレイク」を待つ。 |
| **Greed (55-75)** | **「トレンドフォロー (Ride the Wave)」** <br> 素直に上昇トレンドに乗る局面。利益を伸ばすフェーズ。 |
| **Extreme Greed (>75)** | **「利食い / 警戒 (Take Profit / Caution)」** <br> 相場が過熱しており、急落のリスクが高い。**「AIシナリオは強気だが、センチメントは過熱」** という場合、天井掴みを避けるために**あえて買わない**、または**利益確定（利確）**を行う判断材料になる。 |

---

**結論**:
FinShift Sentiment Indexは、単なる「今の気分」を示すものではなく、**「AIシナリオ（論理）」と「市場心理（感情）」のギャップを可視化し、より精度の高いエントリー・エグジット判断をサポートするツール**です。
